CleanPro este un site web de prezentare destinat unei firme de curățenie din Târgu Mureș, având ca scop promovarea serviciilor oferite și facilitarea contactului rapid cu potențialii clienți.

Scopul proiectului

Scopul site-ului este de a oferi informații clare și accesibile despre serviciile de curățenie, de a crește vizibilitatea firmei în mediul online și de a permite clienților să solicite oferte sau programări într-un mod simplu și eficient.

Structura site-ului (pagini)

Pagina principală (Home) – prezentare generală a firmei, slogan, servicii principale și apel la acțiune

Servicii – descriere detaliată a serviciilor de curățenie (rezidențială, birouri, industrială etc.)

Despre noi – informații despre firmă, experiență și valori

Contact – formular de contact, date de contact și locația firmei

Galerie / Portofoliu (opțional) – imagini cu lucrări realizate

Tehnologii utilizate

HTML5 – pentru structura semantică a paginilor web

CSS3 – pentru stilizare, layout responsive și design modern

JavaScript – pentru interactivitate (formular de contact, animații, validări)
